# Stability-Driven Radiomics Predicts Microvascular Invasion in Hepatocellular Carcinoma on Gadoxetic Acid-Enhanced Magnetic Resonance Imaging

This repository provides partial code for our study on a stability-driven radiomics framework for predicting microvascular invasion in hepatocellular carcinoma using gadoxetic acid-enhanced MRI.

To comply with the current manuscript review and publication process, only part of the code is publicly available at this stage.

A more comprehensive and better-organized version of the code will be uploaded after formal publication of the associated article.

Please refer to the published paper for full methodological details.
